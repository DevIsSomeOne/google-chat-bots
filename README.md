This is a detailed description of every bot in this repository.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------


1. AFK bot
   This Google Chat bot responds to anyone who pings you or says your name with a message letting them know what your AFK
______________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________

2. Anonymous bot
   This bot will blur any images sent in a dm and replace their names with fake ones. Also replaces their profile pictures. This it to make it easier to take scrteenshots when you want to remain anonymous.
______________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________

3. Blank text
   This script is really simple. Running it in console makes it so that you can send an invisible character.
______________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________

4. Ghost bot
   This bot intercepts some information from google so that as long as you don't scroll the other person doesn't see that you saw their message.
______________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________

5. Reply bot
   This bot replys to a specified users message with "lol" or whatever text you chose to send.
______________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________

6. Spam bot
   This bot spams a specified word as fast as you want it to. Going too fast wont send some of the messages.\
______________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________

7. Text counter bot
   This bot will count every single message in your dms with someone. This is a lot nicer than downloading your user logs from Google Takeout.
______________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________

8. Time bot
   This bot sends specified text messages at certain times that the user can custumize.
______________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________

9. Welcome bot
    This bot welcomes new users when they join a space.
